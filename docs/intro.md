# Components
```jsx
Atom is a small, theme-driven React component library built on:

●  React + TypeScript
●  Tailwind (utility classes inside the bundle)
●  Design tokens via CSS variables  (e.g. `--atom-primary`, `--atom-button-bg`)
●  A theme wrapper: `.atom-theme[data-theme="…"]`

The goal: **production-ready, fintech-friendly UI primitives** with a clean theming story and minimal setup 
in consuming apps.
```
## Installation

```jsx
npm install @workokay/atom
# or
yarn add @workokay/atom
```
