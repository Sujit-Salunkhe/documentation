```md
Peer dependencies
    Make sure you have:
        ● react
        ● react-dom
        ● @types/react / @types/react-dom (if using TypeScript)
        ● class-variance-authority (for typing support from cva-based variants)

    ✅ Tailwind is not required in the consuming app for colors to work.
    Atom ships its own atom.css with design tokens and fallbacks for the token-based utility classes.
```
